# 📚 DSA From Scratch

A comprehensive repository for learning **Data Structures and Algorithms** from the ground up.

This repository is designed for beginners and intermediate programmers who want to understand DSA conceptually before solving problems.

## 🎯 Objectives

* Learn Data Structures from first principles.
* Understand Time & Space Complexity deeply.
* Explore the internal implementation of each data structure.
* Visualize algorithms with simple explanations.
* Practice with real coding examples in Java.
* Prepare for coding interviews.

# 📊 Complexity Cheat Sheet

| Complexity | Performance  |
| ---------- | ------------ |
| O(1)       | Constant     |
